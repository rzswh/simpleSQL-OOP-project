## 运行说明

### 编译条件

本程序在Ubuntu 18.04.2 64-bit, g++ 7.3.0下编译通过。
本程序只需要使用make工具即可编译，不依赖于C++标准库之外的库。
make clean需要在Linux下使用（因为用的是rm）。

### 运行条件

本程序在Ubuntu和Windows下都可以正常运行。
